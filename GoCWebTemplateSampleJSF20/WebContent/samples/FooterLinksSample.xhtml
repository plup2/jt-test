GoC Web Template Samples - Footer Links

The footer links consist of the Contact, News and About links located at the bottom of the page. The theme you specify for your site determines which category and links are displayed. The theme also determines which of the categories can be modified by the application. Example in the "GCWeb" theme, only the Contact links can be customized.

Pre-requisite

To override the Default GoC Web Template look & feel, you will have to create a custom bean class that extends the goc.webtemplate.component.jsf.BaseCoreBean class, and then override the various methods made available to alter the look & feel of the web page.

For this particular sample page, we are using the goc.webtemplate.jsf.samplebeans.FooterLinksSample bean class, and the bean name is "footerlinkssamplebean".

The bean must be included at the top of your jsf (xhtml) page as part of the beaninit section that is exposed by the master template page as following:

<ui:define name="beaninit">
	<ui:param name="goctemplateclientbean" value="" />		
</ui:define>

Setting the Contact Us link

The "Contact Us" link at the bottom of the page can be customized by populating the contactLinkUrl variable:

@Override
public void onWebTemplateInitialize() {
//...
    this.setContactLink(new Link("http://travel.gc.ca/"));
//...
}
